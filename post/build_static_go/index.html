编译全静态无依赖GO可执行程序

2019-08-04 276 words 1 min read

Contents

    • 编译命令
    • 依赖库问题
    • Docker 例子

　　Go非常强悍的新兴语言， 未来大数据，云计算的天下，非GO莫属。 编译一个无任何动态链接库依赖的程序是非常有用的，因为你不知道你的程序将会在什么环境上运行。而且结合Docker这个运行平台，静态可执行程序有你意想不到的效果！

编译命令

go install -v -ldflags '-extldflags -static -s -w'

依赖库问题

　　也许这术编译的时候还是会依赖libstdc++.so.6, libgcc_s.so.1,ld-musl-x86_64.so.1. go install -v -ldflags '-linkmode external -extldflags -static -s -w'

    from golang:alpine as builder
Add . .
RUN go build -a -ldflags "-s -w -X 'main.goVersion=$(go version)' -X 'main.date=$(date)'" hello.go
from scratch
copy  --from=builder /go/hello /hello
CMD ["/hello"]
